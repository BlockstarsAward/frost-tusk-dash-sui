# Frost Tusk Dash: Sui Arcade Edition

A fast-paced, 2D side-scrolling platformer game styled after classic arcade runners, optimized for cross-platform play and integrated with the Sui Testnet blockchain architecture.

## 🕹️ Core Gameplay
- **The Mission:** Control the character to run and jump across the stage, collecting spinning gold coins while dodging deadly obstacles.
- **Difficulty Scaling:** Hazards smoothly speed up as your score increases to challenge your reflexes.
- **Adaptive Inputs:** Large touch-screen controls activate automatically on mobile devices, while shifting seamlessly to keyboard mapping (Arrows/WASD/Spacebar) on desktop platforms.

## ⛓️ Web3 & Sui Integration
- **Instant Sandbox Wallet:** Generates a cryptographically valid Sui Testnet wallet address directly inside the browser using the official `@mysten/sui.js` SDK.
- **n1stake Faucet Integration:** Integrated clipboard utility to easily copy your ephemeral address and request free gas tokens from the community-driven n1stake faucet pool.
- **Arcade Economy Loop:** Simulates an authentic on-chain gaming loop by charging a local entry fee of `0.05 SUI` to launch each round.
- **Persistent Leaderboard:** Saves and indexes personal high-scores locally inside the browser memory across updates and refreshes.
